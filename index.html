HTML Fundamentals

Introduction

HTML is the standard markup language for creating web pages. First created by Tim Berners-Lee in 1991, HTML has evolved to become the backbone of the modern web.

Why HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML elements tell the browser how to display the content
  • HTML consists of a set of tags contained within an HTML document, and the associated files typically have either a ".html" or ".htm" extension
  • There are several versions of HTML, with HTML5 being the most recent version

How do websites work?

A typical HTML page looks like this:

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph</p>
<p>This is another paragraph</p>
</body>
</html>

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

Every HTML document starts with this basic structure. The DOCTYPE declares the document type, and the html element is the root element.
